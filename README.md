## Compile it yourself

### Prerequisites
- Download and install Node LTS [16.15.X](https://nodejs.org/it/download/)
- Install [Yarn](https://classic.yarnpkg.com/lang/en/docs/install/)

### Android
- `yarn run android`

### iOS
- `cd ios && pod install`
- `yarn run ios`
